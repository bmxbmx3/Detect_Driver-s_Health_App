# 基于移动互联网的智能车载APP
本项目是参加第七届“中国软件杯”大学生软件设计大赛中“基于移动互联网的智能车载App”赛题的作品，集硬件、安卓端、云服务器于一体，用于实时监测汽车司机的健康状况。
